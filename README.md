Laravel template with breeze modded multiauth (admins and Users).

- Area Utenti
--- Profilo
- Area Admin
--- Dashboard
--- Gestione Utenti

Include:
- Breeze (moddato multiauth)
- kamona/kui sidebar
- Yajra DataTables
- ApexCharts
- Fontawesome-free
- Notyf
- Sweetalert2



Install:

Clone Git

Configure .env file

php artisan migrate

php artisan db:seed

php artisan db:seed --class=CreateUsersSeeder
